## 类型字段

每个品牌可以带一个 `type` 字段：

- `main`：主线品牌。适合大众知名、全国化或区域强知名品牌。
- `local`：地方隐藏款。适合本地有存在感，但不是全国大众品牌的条目。
- `landmark`：茶饮地标。适合甜茶馆、传统奶茶、地方饮品文化，不一定是连锁品牌。

没有写 `type` 时，页面会按 `main` 处理。

## 字段说明

每条品牌数据结构：

```js
{
  id: "唯一英文 id",
  name: "品牌名",
  provinceId: "发源地省份 id",
  city: "发源城市",
  confidence: "A/B/C",
  type: "main/local/landmark",
  tags: ["标签1", "标签2"],
  noteType: "本命气质/偏好注脚/隐藏支线/茶饮地标",
  note: "选为本命时使用的短文案"
}
```

可信度建议：

- `A`：官方信息、招股书、品牌公开资料较稳。
- `B`：公开资料较多，但仍建议上线前复核。
- `C`：区域条目或资料较散，适合保留但需要后续核验。
