# 我的奶茶版图 MVP 规格

实际页面入口是 `index.html`，核心交互逻辑在 `app.js`，品牌与人格文案在 `data.js`。

## 产品定位

一个基于中国奶茶品牌发源地的移动端互动地图。用户在地图上点开省份，选择喝过、想喝、本命 Top 1、常驻 Top 2、心头好 Top 3，最后生成奶茶人格和 9:16 分享图。

## 当前版本重点

- 地图使用固定中国地图贴图，省份点击范围使用已固化的圆角矩形热区。
- 不再包含排版模式；当前坐标和展开卡片尺寸已经写死在 `app.js` 的 `provinceLayout`。
- 视觉从静态彩色地图改为成就系统：省份会根据喝过、想喝、Top 状态逐渐点亮。
- 成就色按喝过数量分级，而不是同色深浅：0 杯白色，未收录灰色，1/2/3/4/5+ 杯使用不同等级色。
- 地图由省份 mask 重绘底色，再叠加原图提取的边界和省名；原始彩色底图不再决定省份颜色。
- 展开卡片尊重已固化位置，但宽高会根据品牌数量自动变大，并在地图边界内回收。
- 品牌分为 `main` 主线品牌、`local` 地方隐藏款、`landmark` 茶饮地标。
- 无品牌地区点开后保持同样的展开卡片样式，显示“尽请期待”。

## 省份颜色等级

| 状态 | 颜色 |
|---|---|
| 暂未收录 | 浅灰 `#d8d8d2` |
| 有收录，0 杯 | 奶白 `#fffdf5` |
| 喝过 1 杯 | 薄荷绿 `#bff1c8` |
| 喝过 2 杯 | 青绿 `#7ddbb4` |
| 喝过 3 杯 | 湖蓝 `#78c7e8` |
| 喝过 4 杯 | 靛蓝 `#7e9ce6` |
| 喝过 5 杯及以上 | 金色 `#f2c94c` |

## 状态规则

- 没选就是没喝过。
- `tried`、`top1`、`top2`、`top3` 都计入喝过。
- 设置新的 Top 1/2/3 时，同一排名之前的品牌自动变成喝过。
- 想喝不参与主人格计算，但参与好奇心/探店倾向和地图黄色提示。

## 成就统计

结果区当前展示：

- 基础款：喝过的主线品牌 / 总主线品牌
- 隐藏款：喝过的地方隐藏款与茶饮地标 / 总隐藏条目
- 产地开图：喝过品牌覆盖的发源地 / 有收录品牌的发源地
- 本命奶茶：当前选择的本命 Top 1 品牌

## 分享图

分享图比例是 9:16 PNG。分享图会同步绘制地图、统计、人格和省份成就色。

## 文案入口

- 页面固定文案：`index.html`
- 品牌、标签、注脚、人格规则：`data.js`
- 交互文案与成就标题：`app.js`
